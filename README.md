# Installation

⚠️ **Warning:** This will overwrite your configs!

Clone repo:

```bash
git clone --recurse-submodules -j8 https://github.com/mbrav/dotfiles
```

Copy dotfiles:

```bash
cp -r ./dotfiles/dotfiles/.*  ~/
```
